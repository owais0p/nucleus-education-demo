# Nucleus Education Demo - Project Instructions

## Tech Stack & Architecture
- **Framework**: Astro (SSG/SSR)
- **Styling**: Tailwind CSS (Vite plugin)
- **Deployment**: GitHub Pages (Base path: `/nucleus-education-demo/`)

## Design System: "Modern Academic Prestige"
A visually striking, elite educational aesthetic characterized by high-fidelity motion and sophisticated spatial composition.

### Typography
- **Display**: `Outfit` (Bold, tracking-tight) for primary headings.
- **Body**: `Plus Jakarta Sans` (Light/Medium) for general content.
- **Editorial**: `Fraunces` (Italic) for elegant accents and narrative emphasis.

### Color Palette
- **Navy**: `#0d1f3c` (Primary background, deep professional tone)
- **Electric**: `#1e90ff` (Primary accent, brand color)
- **Background Details**: Subtle grain texture (`noise-bg`) and glassmorphism (`backdrop-blur-xl`).

## UI Patterns & Components
- **Navbar**: Sticky, frosted-glass effect with uppercase tracking-wide links.
- **Bento Box**: Used for "Why Nucleus" section to create a modern, organized visual hierarchy.
- **Animations**: 
  - Intersection Observer used for staggered `animate-reveal` on scroll.
  - Hover states: Scale transforms and border glows for cards.
- **Logos**: Always use official, high-fidelity SVG paths (e.g., official WhatsApp logo).

## Workflow & Conventions
- **Routing**: All internal links must be prefixed with `/nucleus-education-demo/` due to GitHub Pages hosting.
- **Content Updates**:
  - Admission Session: 2026-27
  - Copyright Year: 2026
- **Contact Forms**: Prioritize direct communication (Phone/WhatsApp) over email.
